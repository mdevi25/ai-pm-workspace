# Personal Command Center (PCC)

## Overview

Personal Command Center is a Streamlit-based web application that serves as a central hub for:

* Projects
* Learning
* AI Tools
* Resume
* Weekly Planning
* AI Experiments

## Tech Stack

* Python
* Streamlit
* GitHub
* Streamlit Cloud
* (Future) OpenAI, LangChain, n8n
